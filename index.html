In this step-by-step tutorial we will create a simple Foosball Game using HTML, CSS, Javascript and JQuery.

NOTE: To get the most out of this series of lessons you should already have basic to intermediate JavaScript/JQuery knowledge.

About the Game:
Foosball (or Table Football) is a tabletop version of soccer in which players turn rods fixed on top of a playing box and attached to miniature figures of players, in order to flick the ball and strike it toward the goal.

In our game, the player will try to make a goal, kicking the ball at a certain angle so that opposing players do not stop it. Note that you can use the field walls to bounce the ball.

Let's define some global variables:

   b) direction_1, direction_2 & direction_3: to define in which direction the players move..
JS / JQuery
var playing = true;
var direction_1, direction_2, direction_3;
To define in which direction the players move, let's use: 
JS / JQuery
direction_1 = Math.floor(Math.random()*500)%2;
direction_2 = Math.floor(Math.random()*500)%2;
direction_3 = Math.floor(Math.random()*500)%2;
A value of 0 will move the player to the left.
A value of 1 will move the player to the right.

There are 3 functions to move the players, called when the game start:
JS / JQuery
move_players_1();
move_players_2();
move_players_3();
As you have already guessed, each function moves the players of each line.
JS / JQuery
function move_players_1() {
   if (playing) {
      left_1 = (parseInt($("#players_line_1 .players:eq(0)").css("left"))+(direction_1===0?-1:1));
      $("#players_line_1 .players:eq(0)").css("left",left_1+"px");
      if (left_1>210) {
         direction_1 = 0;
      }
      if (left_1<105) {
         direction_1 = 1;
      }
      setTimeout(move_players_1, 15);
   }
}

Next: Collisions
Play the Game
Remember the // collision handling comment in the step callback on #ball animation? Now its time to see how we will handle the collision:
JS / JQuery
ball_offset = 
   $(this).offset();
$(".players").each(function() {
   player_offset = $(this).offset();
   if (collided(ball_offset, player_offset)) {
      playing=false;
      $("#ball").stop();
      fail();
      return false;
   }
});
Here we go through all the .players elements using jQuery function each().

For each player we get their absolute left and right (based on the page and not on their container). That's why we use JQuery's offset().

NOTE: The return false; inside each function leaves the each cycle without processing the following elements.

We have a function called collided that returns if the ball and feet of each player match.

JS / JQuery
function collided(b_offset,p_offset) {
   b_top = b_offset.top;
   b_left = b_offset.left;
   p_top = p_offset.top+66;
   p_left = p_offset.left;
   if ((b_left+33)>=p_left &&
      b_left<=(p_left+35) &&
      b_top>=(p_top-20) &&
      b_top<=p_top) {
      return true;
   }
   return false;
}
NOTE: To be honest, this was the part that was most difficult for me. I tried several approaches, but when I thought about using offset, everything became simple.

The first part of the conditional:
JS / JQuery
(b_left+33)>=p_left &&
b_left<=(p_left+35)
checks if the ball is horizontally between the player measures. We add 33 pixels to the ball because that is its width, and we add 35 pixels to the player since that is the width of each player.

The second part of the conditional:
JS / JQuery
b_top>=(p_top-20) &&
b_top<=p_top
checks if the ball is vertically between the player feet. We add 66 pixels to the player since that is the height of each player. We subtract -20px on players top (p_top-20) because we want to check just is the feet of each players collide.

As we have seen in the previous chapters, goal() and fail() functions have been used:
Goal:
JS / JQuery
function goal() {
  var goals = parseInt($("#goals").html())+1;
  $("#goals").html(goals);
  playing = false;
  $("#ball").stop().animate({
    "top":"-30px"
  },{
    duration:100,
    easing:"linear",
    complete: function() {
      $("#goal").fadeTo(10,1,function() {
        $("#goal").stop().animate({
          "background-size":"100%"
        },{
          duration:600,
          easing:"linear",
          complete: function() {
            $("#playagain").show();
          }
        });
      });
    }
  });
}
At the beginning, we increase the value of the div #goal by one. Then we set the variable playing = false, this causes the player's animations to stop.

Finally, we will make the ball (currently with top=0) get inside the goal (top=-30), shows the image goal.png coming from behind, and shows the button #playagain.
Failing
JS / JQuery
function fail() {
  playing = false;
  var ball_top = parseInt($("#ball").css("top"));
  $("#ball").stop().animate({
    "top":(ball_top+50)+"px"
  },{
    duration:400,
    easing:"linear",
    complete: function() {
      $("#playagain").show();
    }
  });
}

First, set the variable playing = false to stop players animations.

Then, we make a small animation to show that the ball bounced (either from the end line or from a player), and shows the #playagain button.

It's time to mention a function named start(). It is called when the user presses the Play or Kick Again button.
JS / JQuery
function start() {
  $("#instructions").hide();
  $("#goal").fadeTo(100,0,function() {
    $("#goal").css("background-size","1%");
  });
  playing = true;
  $("#playagain").html("Kick Again").hide();
  $("#ball,#ball_direction,#deg").show();
  $("#deg").val(0);
  rotate_ball(0);
  $("#ball").css({
    "top":"330px",
    "left":"130px"
  });
  team = Math.floor(Math.random()*500)%2;
  $(".players").each(function() {
    bgp = $(this).css("background-position").split(" ");
    bgp = bgp[0]+" "+(team*-66)+"px";
    $(this).css("background-position",bgp);
  });
  pos = Math.floor(Math.random()*100)+50;
  $("#players_line_2 .players:eq(0)").css({
    "left":(pos)+"px"
  });
  $("#players_line_2 .players:eq(1)").css({
    "left":(pos+140)+"px"
  });
  pos = Math.floor(Math.random()*100)+50;
  $("#players_line_3 .players:eq(0)").css({
    "left":(pos)+"px"
  });
  $("#players_line_3 .players:eq(1)").css({
    "left":(pos+140)+"px"
  });
  $(".players").show();
  direction_1 = Math.floor(Math.random()*500)%2;
  direction_2 = Math.floor(Math.random()*500)%2;
  direction_3 = Math.floor(Math.random()*500)%2;
  move_players_1();
  move_players_2();
  move_players_3();
}
  1. Hide #instructions
  2. Hide #goal
    (the goal alert)
  3. set playing = true
    (players animation enabled)
  4. show #ball, #ball_direction and .slider
  5. set .slider value to 0, and
    rotates #ball_direction to 0
  6. put the #ball in its starting place
    (top:330px, left:130px)
  7. set position-background based
    on team variable
  8. Create variables with random values that will define: the initial positions of the players and the direction in which the players move.
  9. And finally, start the movement of the players.
